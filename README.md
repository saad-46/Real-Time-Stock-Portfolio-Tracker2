# 📈 StockVault Premium (v4.0.0)

StockVault Premium is an institutional-grade stock portfolio tracker built with **100% Pure Java**. It features AI-driven analytics, real-time market simulation, and a high-performance Bloomberg-style UI.

## 🚀 Quick Start

### Prerequisites
- **Java Development Kit (JDK) 17 or higher**
- **Git**

### Installation
1.  **Clone the Repository**
    ```bash
    git clone https://github.com/saad-46/Real-Time-Stock-Portfolio-Tracker.git
    cd Real-Time-Stock-Portfolio-Tracker
    ```

2.  **Run the Application**
    Simply execute the provided batch file to compile and launch the dashboard:
    ```bash
    .\RUN-PREMIUM-DASHBOARD.bat
    ```

## ✨ Key Features
-   **AI Portfolio Rebalancer**: Institutional-grade optimization engine for sector diversification.
-   **Real-Time Market Pulse**: Streaming live feed of portfolio and watchlist data.
-   **Advanced Analytics**: Risk assessment (Sharpe Ratio, Beta) and sector allocation charts.
-   **Bloomberg-Level UI**: High-performance dark theme with sparkline indicators and micro-animations.
-   **Notification Engine**: Native desktop alerts for trade confirmations and price triggers.
-   **Data Export**: Premium HTML/PDF reporting and CSV export.

## 📂 Project Structure
-   `src/`: Core Java source code (Model, View, Service, Database layers).
-   `lib/`: Required library dependencies (JFreeChart, FlatLaf, SQLite, etc.).
-   `docs/`: Detailed feature guides, API setup, and historical project documentation.
